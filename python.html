Console

Code Example Explanantion
print(data) print("hi there") Prints data to console. Data can be a variable, string, integer, or more. You can print almost anything.
varname = data x = 10 Creates a new / sets an existing variable to data. Variables can be any data type, but cannot change. You cannot set a boolean to a string, etc.
input(text) y = input("Enter a number: ") This function pauses the entire program to take an input from the console. It will not continue until the user presses "enter". It will return a string of the typed text. You can set a variable to the typed text. This is commonly used for questions and clicker games.
